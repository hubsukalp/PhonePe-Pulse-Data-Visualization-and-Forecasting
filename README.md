# 📊 PhonePe Pulse Data Visualization & 2026 AI Forecast

## 🚀 Project Overview
An end-to-end data science project that extracts, cleans, and visualizes PhonePe Pulse data. It features a **Random Forest Regressor** to predict transaction trends for 2026.

## 🛠️ Tech Stack
- **Python** (Pandas, Scikit-Learn)
- **Database:** SQLite
- **Dashboard:** Streamlit
- **Visualization:** Plotly

## 📈 Key Results
- **Model Accuracy:** 99.59% R-squared.
- **2026 Forecast:** Projected 22.63% growth in transaction volume for Maharashtra.
- **Key Driver:** Transaction Intensity.
